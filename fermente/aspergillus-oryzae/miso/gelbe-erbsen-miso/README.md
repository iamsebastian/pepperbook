# Gelbe Erbsen-Miso

| Zutat | Menge |
| :--- | :--- |
| Gelbe Erbsen, gekocht  | 3/5 |
| Koji | 2/5 |
| Salz | 4% |

### Erst-Versuch

Bezug-nehmend auf das: 

{% page-ref page="../../koji/perlgraupen-koji.md" %}

$$ Koji := 100g\ \&$$  
$$Gelbe Erbsen := 150g\ \&$$  
$$Salz := 10g $$

Bei 28ºC ist das Ergebnis in etwa 3 Monaten erreicht.
